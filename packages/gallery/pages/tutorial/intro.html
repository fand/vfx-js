VFX-JS Gallery

This gallery is a flat collection of standalone HTML pages used for manual review and visual regression testing of @vfx-js/core.

Layout

Pages live under pages/<kind>/. The top-level directory determines how the page is treated:

  • pages/test/ — visual regression targets; Playwright walks every HTML file here.
  • pages/tutorial/ — onboarding / how-to pages, not asserted.
  • pages/demo/ — technical demos, not asserted.

Per-page scenarios

A page may have a sibling *.scenario.ts file that describes how to drive it (viewport, interactions, multiple shots, etc.): 

// pages/test/foo.scenario.ts
import { defineScenario } from "../../src/runner/scenario";

export default defineScenario([
    { title: "desktop", viewport: { width: 1280, height: 720 } },
    { title: "mobile",  viewport: { width:  375, height: 667 } },
]);

Pages without a scenario file fall back to a default (1280x720, one viewport screenshot, waits for window.__galleryReady === true).
